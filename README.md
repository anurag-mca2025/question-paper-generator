# Question Paper Generator

A C++ program that automatically generates question papers from text files.

## Quick Start

1. **Build:**
   ```
   BUILD.bat
   ```

2. **Run:**
   ```
   question_generator.exe
   ```

3. **Enter file or folder path when prompted:**
   - Single file: `D:\backup\question-paper-generator\DSA.txt`
   - Folder: `D:\backup\question-paper-generator`

## Project Structure

```
question-paper-generator/
├── src/              # Source files (.cpp)
├── *.h               # Header files
├── BUILD.bat          # Build script
├── Makefile           # Make build file
└── README.md          # This file
```

## Features

- Generates MCQs, Fill-in-the-blanks, Short and Long answer questions
- Accepts single .txt files or folders with multiple .txt files
- Automatically processes and scores content
